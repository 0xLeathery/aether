## Tech Debt

### Phase 13: Interactive Demo (2 items)

1. **System message field stripped at deserialization** — `webrtc.ts` sends join notification with `system: true` but `onmessage` handler strips the field during cast. Join message renders as regular chat message instead of system-styled message. Connection status is correctly shown via status bar. Cosmetic only.

2. **Google STUN servers at runtime** — `webrtc.ts` contacts `stun.l.google.com:19302` during demo sessions. These are runtime network requests (not page-load assets), not a SITE-03 violation by strict interpretation, but worth noting for privacy-conscious users.

### Phase 15: Verification Closure (1 item)

3. **Dead NavLink import in MobileMenu** — `MobileMenu.svelte` imports `NavLink` component (line 3) but renders internal links using plain `<a>` tags. Active route highlighting (green underline) absent from mobile menu. Desktop nav correctly uses NavLink. Cosmetic inconsistency.

**Total:** 3 items across 2 phases. All cosmetic/informational. No blockers.
